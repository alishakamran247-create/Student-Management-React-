# 🎓 Student Management App

A React.js Single Page Application (SPA) for managing student records, built as my WAD Assignment 4 at the University of Central Punjab.

## 🚀 Features

- ➕ Add new student records via form
- 📋 Display list of all students
- 🌐 Fetch and display data from external API
- 🔄 Global state management using React Context API

## 🛠️ Technologies Used

- **Framework:** React.js (Vite)
- **Language:** JavaScript (JSX)
- **State Management:** React Context API
- **Styling:** CSS
- **Concepts:** Functional Components, React Hooks, Props, Context

## 📂 File Structure

```
student-management-react/
│
└── src/
    ├── App.jsx                        # Main app component
    ├── App.css                        # App styling
    ├── main.jsx                       # Entry point
    ├── index.css                      # Global styles
    ├── components/
    │   ├── StudentForm.jsx            # Form to add students
    │   ├── StudentList.jsx            # Displays student list
    │   └── ApiData.jsx                # Fetches external API data
    └── context/
        └── StudentContext.jsx         # Global state management
```

## 🚀 How to Run

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

Then open `http://localhost:5173` in your browser.
